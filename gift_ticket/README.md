Gift Ticket templates

Files
- option-1-classic.html — Classic printable ticket layout with left/right columns and QR placeholder.
- option-2-modern.html — Modern, hero-style e-ticket with a prominent Claim button.
- option-3-foldable.html — Two-panel foldable style (front/back) for printing or emailing.

- Placeholders to replace
- LOGO_PLACEHOLDER / assets/* — provide team logo images (recommended sizes: 300x300 PNG or SVG).
- GAME_TITLE_PLACEHOLDER — text like "Home Team vs Opponent".
- DATE_PLACEHOLDER — full date/time and stadium.
- RECIPIENT_NAME_PLACEHOLDER — recipient's name.
- TICKETS_PLACEHOLDER — number of tickets (e.g. "2 tickets").
- TICKET_NUMBER_PLACEHOLDER — optional ticket number for display.
- BARCODE/IMAGE_PLACEHOLDER — optional: if you want a decorative barcode image, place it in `gift_ticket/assets/` and update the HTML.
- GIVER_NAME_PLACEHOLDER — your name or message.

How to use
1. Place provided logos in `gift_ticket/assets/` or update `img src` paths in the HTML.
2. Edit the HTML placeholders with the event and friendly gift details (recipient, tickets count, ticket number, and a short message).
3. Open the HTML file in a browser to preview. Print or save as PDF if desired.

What I need from you (optional)
- Team logos (transparent PNG or SVG), or tell me where to fetch them.
- Exact game date/time, stadium, opponent, and seat details.
- Whether you prefer any color/theme adjustments or a printable-friendly version.
